# Ludo Design System

Ludo 디자인 시스템은 [Ludo 팀](https://github.com/Ludo-SMP)이 UI를 더 빠르고 생산적으로 구축할 수 있도록 도와주는 재사용 가능한 컴포넌트 라이브러리입니다.

## 로컬에서 실행하기

```sh
$ git clone https://github.com/Ludo-SMP/ludo-frontend && cd ludo-frontend
$ yarn
$ yarn run storybook
```

### 빌드하기

```sh
$ yarn run build-storybook
```
